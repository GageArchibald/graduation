
# Gage's Graduation Celebration

A modern web application celebrating graduation from Utah Tech University with a Bachelor's Degree in Information Technology.

## Features

- Responsive design with custom typography
- Built with React and TypeScript
- Elegant UI with Outfit and Playfair Display fonts
- SEO optimized with Open Graph metadata

## Tech Stack

- **Frontend**: React, TypeScript
- **Styling**: CSS with Google Fonts
- **Build Tool**: Vite (implied by tsx setup)

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
/src
    main.tsx      # Application entry point
```
